# LLM 模型对比排行榜

**当前版本：1.1.0**

一个基于 Next.js 的大语言模型对比排行榜工具，帮助用户快速比较不同 LLM 模型的性能指标、价格和规格。

## 功能特性

### 模型搜索
- 支持按模型名称或组织进行搜索
- 实时过滤模型列表
- 支持多关键词搜索

### 模型对比
- 支持选择多个模型进行对比
- 直观的对比列表管理
- 一键添加或移除模型
- 预设模型组快速选择
- 一键清空对比列表

### 详细对比信息
对比表格包含以下信息：
- **基本信息**：模型名称、组织
- **规格参数**：上下文长度、多模态支持
- **价格信息**：输入价格、输出价格（美元/百万 tokens）
- **性能指标**：
  - GPQA 分数（研究生级问答）
  - SWE Bench 分数（软件工程基准测试）
  - MMMU 分数（多学科大学水平理解）
- **发布信息**：发布日期

### 数据来源
所有模型数据来自 [ZeroEval API](https://api.zeroeval.com/leaderboard/models/full)，定期更新以确保信息的准确性和时效性。

### 技术特点
- 基于 Next.js 16 和 React 19 构建
- 使用 Tailwind CSS 实现响应式设计
- TypeScript 类型安全
- 简洁的黑白配色设计
- 显示版本信息和编译时间
