# 更新日志

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2025-12-29

### 新增功能
- 添加清空对比列表功能
- 添加预设模型组
- 显示版本信息和编译时间

### 优化改进
- 移除了对模型对比列表数量的限制

## [1.0.0] - 2025-12-29

### 新增功能
- LLM模型对比排行榜
- 模型搜索功能（支持按模型名称或组织搜索）
- 支持选择多个模型进行对比
- 对比详情展示：
  - 模型名称和组织
  - 上下文长度
  - 多模态支持
  - 输入/输出价格
  - GPQA 分数
  - SWE Bench 分数
  - MMMU 分数
  - 发布日期
- 响应式设计，支持桌面和移动端
